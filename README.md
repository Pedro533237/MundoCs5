# MundoCs5

Deterministic Fabric worldgen mod for **Minecraft 1.21.11** that creates a circular “pizza/donut” overworld around `(0, 0)`.

## Layout

- **Center island:** `mushroom_fields`, ~54 block radius.
- **Inner ocean ring:** from ~54 to ~234 blocks, with south-side lukewarm water, north-side cold water, and scattered mini-islands.
- **Main biome ring:** from ~234 to ~864 blocks, arranged clockwise from the south using the requested handcrafted biome order.
- **Outer ocean:** from ~864 to ~2048 blocks, transitioning from warm south waters through temperate mid-ocean into cold/frozen northern waters.
- **Fallback:** after ~2048 blocks, the biome source returns to vanilla overworld generation.

## Implemented biome order

Clockwise from the south:

1. Badlands / Wooded Badlands / Plateau / Eroded Badlands
2. Savanna / Savanna Plateau / Windswept Savanna
3. Desert
4. Sandy + Stony mountain transition
5. Cherry Grove / Forest / Flower Forest
6. Plains with two embedded `stony_peaks` landmarks near the outer edge
7. Taiga / Old Growth Taiga
8. Snowy Taiga
9. Snow mountains (`snowy_plains`, `frozen_peaks`, `jagged_peaks`)
10. Forest band with blended `pale_garden` and small dark forest patches
11. Swamp / Mangrove Swamp
12. Dense Dark Forest strip
13. Jungle / Sparse Jungle / Bamboo Jungle

## Rivers and coastlines

- Procedural river corridors cut radially across the land ring.
- Cold sectors switch those corridors to `frozen_river`.
- Beaches are temperature-aware (`beach` vs `snowy_beach`).
- Coastlines and biome seams use deterministic organic noise instead of hard straight borders.

## Usage

1. Build the mod with Gradle.
2. Install it in Fabric for Minecraft 1.21.11 / Java 21.
3. Create a world with the `mundocs5:pizza_world` preset.
4. Spawn near `(0, 0)` to experience the donut layout immediately.

## Notes

- Terrain shape is still generated by Minecraft's overworld noise settings, so this mod controls **biome placement** and biome-driven flavor rather than replacing the entire chunk generator.
- The layout is deterministic because the biome source uses a fixed `layout_seed`.
